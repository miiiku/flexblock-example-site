在SCSS中定义了一个变量，但是在CSS中使用SCSS中定义的变量无效:

```scss
$color: #f00;

:root {
  --text-color: $color; /** 无效 */
}
```


使用`#{}`Interpolation插值来解决此问题:



```scss
$color: #f00;

:root {
  --text-color: #{$color};
}
```

参考文档：[https://sass-lang.com/documentation/interpolation](https://sass-lang.com/documentation/interpolation)
